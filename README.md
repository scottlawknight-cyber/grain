# بوت الحجز التلقائي - نظام حجز الحبوب العراقي 🌾

## المميزات الكاملة
- ✅ تسجيل دخول عبر تلجرام (مع حل مشكلة النوافذ المنبثقة)
- ✅ قوائم منسدلة تلقائية: المحافظات → المديريات → الشعب → السايلوات
- ✅ البحث عن الفلاح من لائحة الموقع مباشرة (Select2 API)
- ✅ مراقبة مستمرة 24/7 لتوفر الحصص
- ✅ حجز تلقائي فوري عند توفر حصة
- ✅ واجهة رسومية PyQt6 كاملة باللغة العربية

## التثبيت

```bash
pip install -r requirements.txt
```

## التشغيل

```bash
python bot_hajz.py
```

## طريقة الاستخدام

1. **تسجيل الدخول**: اضغط "تسجيل الدخول (تلجرام)" وأكمل العملية في المتصفح
2. **إضافة معاملة**: 
   - اضغط "تحميل المحافظات" لجلب القوائم من الموقع
   - اختر المحافظة → المديرية → الشعبة → السايلو (تلقائياً)
   - ابحث عن الفلاح واختره من اللائحة
   - احفظ المعاملة
3. **تشغيل المحرك**: يعمل على مدار الساعة ويحجز تلقائياً عند توفر حصة

## الإصلاحات المطبقة
1. **تلجرام**: لا يغلق المتصفح حتى التأكد من كوكي المصادقة الحقيقي
2. **الفلاح**: يُسحب من لائحة الموقع عبر API (وليس كتابة يدوية)
3. **المنبثقات**: تُفتح في نفس التبويبة بدلاً من نافذة خارجية تنهار البرنامج
